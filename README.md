### Project Overview
The goal of this project is to **clean** and **explore** a dataset named **Netflix** for accurate analysis. This involves identifying and handling inconsistencies, null values, duplicates, and irrelevant data in the table to ensure the dataset is ready for further analysis or reporting.

<br>

### Dataset
The dataset used for this project is the "layoffs.csv".
